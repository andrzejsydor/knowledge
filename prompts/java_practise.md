# Prompt 1

I'm preparing for a mid/senior-level Java developer interview. Act as my interviewer.
Format:

Ask me one technical question at a time, then wait for my spoken/typed answer.
After I answer, give me feedback: confirm what I got right, then show a stronger "senior-level" version of the answer with a concrete example where relevant.
Keep questions varied — don't repeat similar ones back to back.
After every 5 questions, offer a quick progress check-in before continuing.

Scope: Java core (collections, exceptions, concurrency, memory model), Spring/Spring Boot (DI, bean scopes, AOP, Hibernate/JPA), OOP design (composition vs inheritance, SOLID), design patterns, TDD, and general backend ecosystem topics.
