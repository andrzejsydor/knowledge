## [Constraint Composition](https://www.baeldung.com/java-bean-validation-constraint-composition)

JSR 380: Bean Validation 2.0

> **Note:** Spring Boot 3+ uses `jakarta.validation` (Jakarta EE 9+). Older versions use `javax.validation`.

Reference: [Baeldung — javax/Jakarta Validation](https://www.baeldung.com/javax-validation)

Validation on type arguments:

```java
List<@NotBlank String> preferences;
```

With `Optional`:

```java
private LocalDate dateOfBirth;

public Optional<@Past LocalDate> getDateOfBirth() {
    return Optional.of(dateOfBirth);
}
```

## Spring

### Spring Proxy

- **JDK Dynamic Proxy** — created based on the interfaces implemented by the target class. Built into the JDK.
- **CGLIB Proxy** — generates subclasses (proxies) via inheritance. Used when the target class does not implement any interfaces.

| Proxy Type            | Usage Condition                            | Description                          |
|-----------------------|--------------------------------------------|--------------------------------------|
| JDK Dynamic Proxy     | Class implements at least one interface     | Interface-based proxy built into JDK |
| CGLIB Proxy           | Class does not implement any interface      | Proxy generated by CGLIB library     |

### Spring MVC Test — Page Response

<details>
<summary>Full example: Controller, Test, and RestPageImpl helper</summary>

**Controller + Entity + Repository**

```java
package com.example.demo;

import lombok.Data;
import lombok.RequiredArgsConstructor;
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.data.jpa.repository.JpaRepository;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

import jakarta.persistence.Entity;
import jakarta.persistence.GeneratedValue;
import jakarta.persistence.GenerationType;
import jakarta.persistence.Id;

@RequiredArgsConstructor
@RestController
class ProductController {
    private final ProductRepository repository;

    @GetMapping("/products")
    Page<Product> all(final Pageable pageable) {
        return repository.findAll(pageable);
    }
}

interface ProductRepository extends JpaRepository<Product, Long> {
}

@Data
@Entity
class Product {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    Long id;
    String name;
}
```

**Integration Test**

```java
package com.example.demo;

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.junit.jupiter.api.Test;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.boot.test.autoconfigure.web.servlet.AutoConfigureMockMvc;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.data.domain.Page;
import org.springframework.test.web.servlet.MockMvc;
import org.springframework.test.web.servlet.request.MockMvcRequestBuilders;

import static org.junit.jupiter.api.Assertions.assertEquals;

@SpringBootTest
@AutoConfigureMockMvc
class ProductControllerTest {

    @Autowired
    ProductRepository repository;

    @Autowired
    MockMvc mvc;

    @Test
    void filterTest() throws Exception {
        repository.save(getEntity());

        final String content = mvc.perform(MockMvcRequestBuilders.get("/products"))
                .andReturn()
                .getResponse()
                .getContentAsString();

        final Page<Product> page = new ObjectMapper()
                .readValue(content, new TypeReference<RestPageImpl<Product>>() {});

        assertEquals("product1", page.getContent().get(0).getName());
    }

    private Product getEntity() {
        final Product product = new Product();
        product.setName("product1");
        return product;
    }
}
```

**RestPageImpl — Jackson-deserializable Page**

```java
package com.example.demo;

import com.fasterxml.jackson.annotation.JsonCreator;
import com.fasterxml.jackson.annotation.JsonProperty;
import com.fasterxml.jackson.databind.JsonNode;
import org.springframework.data.domain.PageImpl;
import org.springframework.data.domain.PageRequest;
import org.springframework.data.domain.Pageable;

import java.util.ArrayList;
import java.util.List;

public class RestPageImpl<T> extends PageImpl<T> {

    @JsonCreator(mode = JsonCreator.Mode.PROPERTIES)
    public RestPageImpl(@JsonProperty("content") final List<T> content,
                        @JsonProperty("number") final int number,
                        @JsonProperty("size") final int size,
                        @JsonProperty("totalElements") final Long totalElements,
                        @JsonProperty("pageable") final JsonNode pageable,
                        @JsonProperty("last") final boolean last,
                        @JsonProperty("totalPages") final int totalPages,
                        @JsonProperty("sort") final JsonNode sort,
                        @JsonProperty("first") final boolean first,
                        @JsonProperty("numberOfElements") final int numberOfElements,
                        @JsonProperty("empty") final boolean empty) {
        super(content, PageRequest.of(number, size), totalElements);
    }

    public RestPageImpl(final List<T> content, final Pageable pageable, final long total) {
        super(content, pageable, total);
    }

    public RestPageImpl(final List<T> content) {
        super(content);
    }

    public RestPageImpl() {
        super(new ArrayList<>());
    }
}
```

### Failsafe

[Failsafe](https://failsafe.dev/) is a lightweight, zero-dependency library for handling failures in Java 8+. It has a concise API for everyday use cases and flexibility for everything else.

```java
RetryPolicy<Object> retryPolicy = RetryPolicy.builder()
    .handle(ConnectException.class)
    .withDelay(Duration.ofSeconds(1))
    .withMaxRetries(3)
    .build();

Failsafe.with(retryPolicy).run(() -> connect());

Connection connection = Failsafe.with(retryPolicy).get(() -> connect());
```
