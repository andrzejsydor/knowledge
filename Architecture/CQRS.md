# #CQRS 




CQRS - Command Query Responsibility Segregation

- Command, Command Handler, Query, Query Handler




PROS:

 - Scalability

 - Flexibility

 - Improved Maintability

 - Enhanced Security

CONS:

 - Increased Complexity

 - Learning Curve

Eventual Consistency
